Cosmetic release: BingWallpaper now ships with a proper app icon.

## What's new since v0.7.1
- **New app icon.** BingWallpaper finally has a real icon — a sunrise landscape framed as a photo — so it looks at home in the Dock, Finder, and Launchpad instead of the generic placeholder. Provided at all standard macOS sizes (16–1024 px).

No functional changes — the wallpaper fetching, notifications, and settings all behave exactly as in 0.7.1.

## Install
1. Download **BingWallpaper_v0.7.2.pkg** below and run it (installs to `/Applications` and launches the app).
2. **Gatekeeper note:** this build is signed ad-hoc (not notarized), so macOS may warn that the developer can't be verified. If so, **right-click the app → Open → Open**, or run:
   `xattr -dr com.apple.quarantine /Applications/BingWallpaper.app`

Or via Homebrew: `brew upgrade --cask bingwallpaper`.

Requires macOS 14 (Sonoma) or later.
