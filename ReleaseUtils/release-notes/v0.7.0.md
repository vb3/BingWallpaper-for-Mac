Adds optional notifications so you know when your wallpaper changes — with a new Settings toggle to turn them on.

## What's new since v0.6.1
- **Wallpaper-change notifications** — when BingWallpaper downloads a new image and sets it as your wallpaper, it can now post a notification showing the image's title and credit, along with a thumbnail.
- **New "Notify when wallpaper changes" setting** — off by default. Turn it on in Settings and macOS will ask for notification permission; if you've previously denied it, BingWallpaper points you to System Settings instead of silently doing nothing.
- **No notification spam** — you get a single notification for the newest wallpaper per update, and re-downloaded older images never trigger a duplicate. Notifications only fire after the wallpaper is actually set.

## Install
1. Download **BingWallpaper_v0.7.0.pkg** below and run it (installs to `/Applications` and launches the app).
2. **Gatekeeper note:** this build is signed ad-hoc (not notarized), so macOS may warn that the developer can't be verified. If so, **right-click the app → Open → Open**, or run:
   `xattr -dr com.apple.quarantine /Applications/BingWallpaper.app`

Or via Homebrew: `brew upgrade --cask bingwallpaper`.

Requires macOS 14 (Sonoma) or later.
